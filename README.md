# ConsoleForBeginners_GUI
## Описание
Данное приложение, созданное с использованием **Qt** версии 6 является простой обёрткой для выполнения каких-либо команд, предназначенных для терминальной оболочки bash (или ещё какой-либо)
в маленьком графическом интерфейсе, кому так может быть проще.
Оболочки для работы
> bash
